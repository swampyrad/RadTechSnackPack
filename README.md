# RadTechSnackPack
A food ration addon for Hideous Destructor.
Includes a variety of food items you can add to your loadouts. 
Foods without codes can only be found in random spawns.

Loadout Codes:
- "drb" - D Ration Bar
- "lgn" - Logan Bar
- "twx" - Diagonal Twix
- "brg" - Trekking Burger
- "chx" - Cereal Ration
- "glg" - Thorn Glognut
- "smk" - "Snusk Juice"
- "ppt" - Pop-Tart
- "mch" - Munchly Meal
- "lmn" - Charged Lemonade
(Requires Ugly As Sin)
